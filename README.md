## Installation

Clone the repository in your HOME directory (this is important!):

    $ git clone --recursive git://github.com/davidjegat/dotfiles.git ~/.dotfiles

Go to the cloned repository

    $ cd ~/.dotfiles

Execute the setup script:

    $ ./setup.sh
